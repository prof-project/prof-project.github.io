Frequently Asked Questions

What is PROF?

PROF stands for PRotected Order Flow. It is a system designed to protect users from harmful MEV extraction, in light of profit-maximizing nature of validators.

How can I learn more about PROF?

You can read our paper or check out our blog post.

If you have more questions, email one of us.

Where can I find the source code?

The source code is available in the prof-project GitHub organization.
